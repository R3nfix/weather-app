# ⛅ Weather App

### [EN]

Weather App is a simple HTML, CSS, and JavaScript weather web app that uses the free OpenWeatherMap API.

### Quick Start

1. Get a free API key from [https://openweathermap.org/](https://openweathermap.org/)
* Register on the website
* Get a free API key (it takes 5-20 minutes to generate)
2. Paste the key into your project
* In the `script.js` file, replace the current value in the line `const apiKey = 'YOUR-API-KEY';` with the key you just received.
3. Done! Run the app.
* Use Live Server (VS Code extension) to launch.
* Otherwise, launch by directly opening the index.html file in a browser (Live Server is recommended).

### Security

Keep your API key strictly for use on your computer. When using it in real projects or on GitHub, don't leave it publicly accessible,
as this is unsafe.

### Stack

* HTML5 (BEM)
* CSS3
* JavaScript

### [RU]

Weather App - это простое веб-приложение на HTML, CSS и JavaScript для просмотра погоды, которое  
использует бесплатный API от OpenWeatherMap. 

### Быстрый запуск приложения

1. Получите бесплатный API-ключ на сайте [https://openweathermap.org/](https://openweathermap.org/)
   * Зарегистрируйтесь на сайте
   * Получите бесплатный API-ключ (для генерации потребуется 5-20 минут)
2. Вставьте полученный ключ в проект
   * В файле `script.js` замените в строке `const apiKey = 'YOUR-API-KEY';` текущее значение на ваш только что полученный ключ
3. Готово! Запустите приложение.
   * Для запуска используйте Live Server (расширение в VS Code)
   * Иначе используйте запуск через прямое открытие файла index.html в браузере (рекомендуется использовать Live Server)

### Безопасность

Оставляйте ваш API-ключ строго для использования на своем компьютере. При использовании в реальных проектах и на GitHub, не стоит оставлять ключ в открытом доступе,  
так как это небезопасно.
